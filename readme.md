Simple, user friendly bot, designed to help discord PR managers and owners!

## Launch

1. Create `.env` file in base dir, basing on [template.env](./template.env) file.
2. Build and run
    (Requires docker 28.5.1)
    ```sh
    docker compose build
    docker compose up
    ```
